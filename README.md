# Emily-Shop

## 项目背景

本项目为化妆品交平台，主要提供功能为支持用户的网上购物，同时实现对用户行为及其他信息的系统化管理。项目使用Vue+SpringBoot+Hibernate技术，实现对数据库的存取、修改等操作。


## 项目配置
阿里云服务器Windows Server2012，Sringboot打包为jar包，Nginx配置vue前端部署。
